# KnitForThat

An e-commerce storefront for a small-batch, handcrafted sweater brand. Built with React, Vite, and Tailwind CSS.

## Projects

### [KnitForThat Landing Page](./KnitForThat)

The customer-facing landing page featuring product listings, brand story, and contact info.

- **Stack:** React 19 + Vite 7 + Tailwind CSS v4
- **Setup:** `cd KnitForThat && npm install && npm run dev`
